================================
 IRI Data Library Documentation
================================

The `IRI Data Library <http://iridl.ldeo.columbia.edu>`_ is a data repository and analysis tool created at the `International Research Institute for Climate and Society (IRI) <http://iri.columbia.edu>`_, focused primarily on data related to Earth's climate and the impact of climate on human populations. This documentation describes the software that powers the Data Library services. While the IRI site is the largest installation of this software, it is also used at dozens of other institutions around the world. It is freely available under the MIT license.
